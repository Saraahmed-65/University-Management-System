  University Management System
A comprehensive University Management System built using Python GUI integrated with a relational database. This system includes 
role-based access control, login authentication, attendance tracking, course management, office hours scheduling, summer course management, and more. It handles different user roles including
Dean, Professors, Teaching Assistants, and Students, with each role having specific access permissions.

---
## üöÄ Features
- Login & Authentication: Secure login for all user roles (Dean, Professor, TA, Student)  
- Role-Based Access Control: Each user sees only what they are allowed to access  
  - Dean: Full access to all users and data  
  - Professor: Access to their students and related courses  
  - TA: Access to assigned courses and students  
  - Student: Access to own records and enrolled courses  
- Attendance Management: Track student attendance  
- Course Management: Add, update, and track courses, including summer courses  
- Office Hours Scheduling: Manage office hours for Professors and TAs  
- Media & Resources: Additional screenshots, diagrams, and files on Google Drive
